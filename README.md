# Advanced-Machine-Learning-project
This project builds an advanced machine‑learning pipeline to predict used‑vehicle prices using the AutoTrader dataset.
It extends beyond basic modelling and demonstrates:

Automated feature selection

Tree‑based ensemble methods

Model ensembling (Voting & Stacking)

SHAP explainability

Permutation importance

Dimensionality reduction (PCA)

Polynomial regression & GLM

This project showcases applied ML engineering skills, including preprocessing pipelines, model comparison, and interpretability.
